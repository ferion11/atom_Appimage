# atom_Appimage

>  Atom Appimage for linux 64bits

####  1- Download "atom-VERSION-x86_64.AppImage"
####  2- Make executable:
```
$ chmod +x atom-VERSION-x86_64.AppImage
```
####  3- Run it:
```
$ ./atom-VERSION-x86_64.AppImage
```
